<h1>Expensify Client</h1>

<p>Link to web page: https://expensify-jordan.herokuapp.com<p>

<h2>Description</h2>

<h2>Technologies</h2>
    <ul>Scripting Language: Node.js</ul>
    <ul>Framework: Express</ul>
    <ul>Authentication: Passport, JWT</ul>
    <ul>Database: MySQL</ul>
